# Генератор надёжных паролей

## Описание

Генератор криптографически надёжных паролей с графическим интерфейсом на Tkinter.

## Возможности

- Генерация паролей длиной от 12 символов
- Использование заглавных и строчных букв, цифр и спецсимволов
- Автоматическая проверка надёжности
- Копирование в буфер обмена
- Сохранение в текстовый файл
- Просмотр/скрытие пароля
- Информативное окно приветствия с инструкциями

## Требования

- Linux (Fedora/RHEL/CentOS)
- Python 3.6+
- Tkinter
- 50 MB свободного места

## Установка

```bash
# Установка RPM пакета
sudo rpm -i password-generator-1.0-1.x86_64.rpm

# Или с автоматическим разрешением зависимостей
sudo dnf install ./password-generator-1.0-1.x86_64.rpm
